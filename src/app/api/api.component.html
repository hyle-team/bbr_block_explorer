API Documentation
How to use
API ENDPOINT
https://explorer.boolberry.com/api/
URL Request Format
https://explorer.boolberry.com/api/{{ '{' }}method}/{{ '{' }}param1{{ '}' }}/{{ '{' }}param2{{ '}' }}
Request current coin stats
Method:
get_info
Example:
https://explorer.boolberry.com/api/get_info/4294967295
JSON Response
Request blocks (offset and count)
Method:
get_blocks_details
Example:
https://explorer.boolberry.com/api/get_blocks_details/{{ '{' }}:offset{{ '}' }}/{{ '{' }}:count{{ '}' }}
JSON Response
Request a given block by hash
Method:
get_main_block_details
Example:
https://explorer.boolberry.com/api/get_main_block_details/{{ '{' }}:hash{{ '}' }}
JSON Response
Request Alt-blocks (offset and count)
Method:
get_alt_blocks_details
Example:
https://explorer.boolberry.com/api/get_alt_blocks_details/{{ '{' }}:offset{{ '}' }}/{{ '{' }}:count{{ '}' }}
JSON Response
Request a given Alt-block by hash
Method:
get_alt_block_details
Example:
https://explorer.boolberry.com/api/get_alt_block_details/{{ '{' }}:hash{{ '}' }}
JSON Response
Request transaction from the pool
Method:
get_pool_txs_details
Example:
https://explorer.boolberry.com/api/get_pool_txs_details
JSON Response
Request brief information transactions from the pool
Method:
get_pool_txs_brief_details
Example:
https://explorer.boolberry.com/api/get_pool_txs_brief_details
JSON Response
Request IDs for all txs from the pool
Method:
get_all_pool_tx_list
Example:
https://explorer.boolberry.com/api/get_all_pool_tx_list
JSON Response
Request a given transaction by hash
Method:
get_tx_details
Example:
https://explorer.boolberry.com/api/get_tx_details/{{ '{' }}:tx_hash{{ '}' }}
JSON Response
